# Bachelor of Computer Science

I earned my Bachelor of Computer Science from the University of Manitoba, where I built a strong foundation in software development, computer systems, database design, and software engineering principles. The program emphasized both theoretical computer science concepts and practical application through programming assignments, collaborative projects, and software design.

Throughout my studies, I developed problem-solving skills and learned how to approach software development with an emphasis on maintainability, scalability, and efficient system design. These principles continue to guide the way I architect applications and write production-quality code.

## Database Systems

One of my primary areas of study was database design, where I completed both introductory and advanced database courses. These courses covered relational database theory, normalization, SQL, query optimization, indexing strategies, transactions, and database modeling.

Working with relational databases strengthened my understanding of designing efficient schemas, maintaining data integrity, and developing applications that interact reliably with large datasets. These concepts have proven invaluable in my professional work developing business applications and content management systems.

## Software Engineering

Software Engineering coursework introduced industry-standard development practices for designing and building large software systems. I gained experience with software architecture, object-oriented design, requirements analysis, version control, testing methodologies, documentation, and collaborative development.

These courses reinforced the importance of writing clean, maintainable code and designing software that can evolve as project requirements grow.

## Artificial Intelligence

My Artificial Intelligence coursework provided an introduction to intelligent systems and algorithmic problem solving. Topics included search algorithms, knowledge representation, heuristic approaches, and foundational machine learning concepts.

The course strengthened my analytical thinking and demonstrated how computational techniques can be applied to solve complex real-world problems.

## Computer Graphics

I completed two Computer Graphics courses that explored the principles behind rendering, animation, and interactive visual computing. Coursework covered topics such as geometric transformations, lighting, 2D and 3D graphics, animation techniques, and the mathematics used to create graphical applications and video games.

These courses provided valuable insight into how graphics engines render and animate objects in real time, while also strengthening my understanding of linear algebra and spatial reasoning. The concepts I learned have carried over into my personal projects, where I've incorporated technologies such as Three.js and React Three Fiber to build interactive 3D experiences and data visualizations for the web.

## Continuing Growth

While my degree provided a strong theoretical foundation, I have continued to expand my skills through professional experience and personal projects. Applying concepts learned in the classroom to real-world applications has strengthened my understanding of full-stack development, responsive design, database architecture, and modern web technologies, allowing me to continually grow as a software developer.
