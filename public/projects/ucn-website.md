# University College of the North Website

As the Web Developer for the University College of the North (UCN), I led the development of the institution's new website, modernizing its design, improving content management workflows, and enhancing the overall user experience. The project involved building a custom WordPress solution tailored to the needs of multiple departments while ensuring the website remained scalable, responsive, accessible, and easy for staff to manage.

## Custom Block Theme

I developed a fully custom WordPress block theme designed specifically for UCN's content and branding requirements. The theme utilizes custom post types to manage key content, including Programs, Events, Faculty Members, Registrar's Office information, News Releases, and Campus Locations.

Using Advanced Custom Fields (ACF) relationships and category-based associations, I created dynamic connections between related content, allowing program pages to automatically display relevant events, news, and additional resources. This approach reduced duplicate content management while improving navigation and helping users discover related information more easily.

The theme was built with a mobile-first approach, ensuring layouts adapt seamlessly across desktop, tablet, and mobile devices. Throughout development, I also prioritized accessibility by following WCAG best practices, implementing semantic HTML, proper heading structures, keyboard-friendly navigation, sufficient colour contrast, and meaningful ARIA attributes where appropriate. These improvements help ensure the website is usable by as many visitors as possible while supporting long-term compliance with accessibility standards.

## Custom Blocks

To provide content editors with greater flexibility, I developed a suite of custom React-based Gutenberg blocks tailored to the university's needs. These include reusable components such as hero sections, content sections, program filters, call-to-action sections, and several additional custom blocks used throughout the website.

Each block was designed to be responsive, accessible, and highly configurable, allowing staff to create engaging page layouts without writing code. By building reusable components, design updates and styling changes can be applied consistently across the entire website, improving maintainability while reducing development time for future enhancements.

## Search Engine Optimization (SEO)

Search engine optimization was a key focus throughout the project. I optimized page titles, meta descriptions, heading structures, internal linking, and overall site architecture to improve search engine visibility and create a better user experience.

I also developed dedicated landing pages for each academic program to target relevant search keywords, increasing opportunities for prospective students to discover UCN through organic search. Working alongside the marketing team, I provided recommendations for content optimization and backlink strategies to strengthen the university's online presence and improve search performance.

By combining custom development, responsive design, accessibility best practices, and technical SEO, the new website provides a modern, scalable platform that delivers an improved experience for both visitors and content editors while supporting the university's long-term digital goals.
